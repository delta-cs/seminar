# Syntaxe množin v Pythonu

Python nabízí implementaci množiny s názvem `set`.

## Vytvoření

```markdown
words = set()
```

## Vložení prvku: `add`

```python
words.add("book")
```

## Vyhledání prvku: `in`

```python
if "book" in words:
    # prvek nalezen
```

## Odstranění prvku: `remove`

```python
words.remove("book")
```

## Délka: `len`

```python
wordsCount = len(words)
```
