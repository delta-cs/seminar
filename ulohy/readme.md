# Úlohy

Složka úlohy vždy obsahuje

- zadání
- podsložku `reseni`
    - text s obecným vysvětlením algoritmu
    - kódová řešení v různých jazycích s případnými komentáři
- podsložku `testy`
    - testové vstupy a očekávané výstupy, kterými musí řešení projít při odevzdání
