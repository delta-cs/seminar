# Pozpátku

<img src="cover.webp" height="200" alt="ilustrace"/>

Na vstupu programu bude

- celé číslo `n` (od 0 do 10 000),
- prázdný řádek,
- `n` celých čísel (od -10 000 do 10 000).

Úkolem je vypsat tato čísla v opačném pořadí, než v jakém přišla na vstup.

## Příklad

### Vstup

```
4

5
-3
1
2
```

### Výstup

```
2
1
-3
5
```
